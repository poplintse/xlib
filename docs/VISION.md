# XLib 产品愿景

XLib 的目标是提供在 Android、iOS 和未来 macOS 上一致、可靠的原生 TXT 阅读体验，同时保持离线优先、精确阅读位置和大文件低内存能力。

## 核心原则

1. 阅读位置以原始文件绝对字节 offset 为权威，百分比只用于展示和粗粒度跳转。
2. 客户端离线时功能完整；同步是可选能力，不能成为阅读前置条件。
3. 不上传书名、正文、本地路径或原始 TXT 文件。
4. 各平台使用原生界面，共享的只是经验证的平台无关核心和 HTTP 合同。
5. 构建、测试和发布必须可重复；普通构建不能修改版本或源文件。

完整的阅读器交互和验收规范见 [features/reader.md](features/reader.md)。
